NewsScrape - A Python application to scrape The New York Times and The Guardian APIs and return a TSV of results.

Structure:
Application: A simple program for gathering at most 10 articles from each source and sorting them in either ascending or descending order of publication date
NYT: A more robust program to gather as many articles as possible from the NYT API, up to 1000 results, on as many queries as the user chooses.
The_Guardian: A more robust program to gather all articles possible from The Guardian API on as many queries as the user chooses
